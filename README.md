# clarity.tk-v1-mapper-fixed
A binary version of the old clarity.tk cheat injected directly into the game.

The cheat was cracked by Qhide & yovimi

> [!IMPORTANT]
> The cheat may occasionally cause crashes during gameplay.

## Compiling
- Build "clarity fixer" as x86, the game itself is 32-bit.
- Build "clarity steam" as x64, since it runs under the 64-bit Steam process.
